# Agent Tools

This directory contains tools that agents can use to perform actions.

## Planned Tools:

### Retrieval Tools
- Vector search
- Keyword search (BM25)
- Hybrid search
- RAPTOR hierarchical retrieval
- Multi-query retrieval

### Database Tools
- Query bills by status/date
- Fetch representative info
- Get voting records
- Search proceedings

### API Tools
- Parliament website scraper
- External API integrations

### Processing Tools
- Document chunking
- Metadata extraction
- Summary generation

Each tool follows the LangChain tool interface.
