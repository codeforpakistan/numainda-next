# Numainda Web App

Main Next.js web application for Numainda v2

## Development

```bash
npm run dev
```

Runs on http://localhost:3000

## Building

```bash
npm run build
npm run start
```

## Testing

```bash
npm test
npm run test:watch
```

## Type Checking

```bash
npm run typecheck
```

## Linting

```bash
npm run lint
npm run lint:fix
```
